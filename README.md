# 🌦️ Weather Forecast Web Application

A simple and elegant **Weather Forecast App** built using **HTML, CSS, and JavaScript**, integrated with the **OpenWeather API** to provide real-time weather updates for any city in the world.

**DEPLOY LINK-->*https://patrarahul5678.github.io/Weather-Forecast/*
## 🚀 Features

- 🌍 Search weather by city name  
- 🌡️ Displays temperature, feels-like temperature, humidity, pressure, and wind speed  
- 📅 Shows current date and city name dynamically  
- ☀️ Responsive and clean UI  
- ⚡ Fetches live data using the OpenWeatherMap API  

---

## 🛠️ Tech Stack

- **Frontend:** HTML, CSS, JavaScript  
- **API Provider:** [OpenWeatherMap API](https://openweathermap.org/api)

## ⚙️ Setup and Installation

Follow these steps to run the project locally:

1. **Clone the repository**
   ```bash
   git clone https://github.com/patraRahul5678/Weather-forecast.git

2. **Navigate to the project folder**
  ```
  cd weather-forecast

  ```
3. **Get your OpenWeather API key**
   Go to OpenWeather API
   Sign up and get your API Key
5. **Add your API Key in script.js**
   ```
   const API_KEY = "your_api_key_here";

   ```
7. **Open the app**
Open index.html in your browser
Type any city name and click Search
