# 🌦️ Weather Forecast Web Application

A responsive weather forecasting application built using **HTML, CSS, and JavaScript**, integrated with the **OpenWeather API** to provide real-time weather updates for cities around the world.

---

## 🌐 Live Demo

👉 **[View Live Website](https://patrarahul5678.github.io/Weather-Forecast/)**

---

## 📖 About the Project

This project was built as part of my journey to learn modern JavaScript and understand how frontend applications interact with external APIs.

The application fetches real-time weather information from the OpenWeather API and displays detailed weather conditions for any city entered by the user.

Through this project, I gained practical experience working with API requests, asynchronous JavaScript, JSON data handling, and dynamic UI updates.

---

## 🚀 Features

* 🌍 Search weather by city name
* 🌡️ Real-time temperature information
* 🤗 Feels-like temperature
* 💧 Humidity details
* 🌬️ Wind speed tracking
* 📊 Atmospheric pressure information
* 📅 Dynamic date display
* 📍 Current city information
* ⚡ Live weather updates using OpenWeather API

## ⚙️ Setup and Installation

### Clone the Repository

```bash
git clone https://github.com/patraRahul5678/Weather-forecast.git
```

### Navigate to the Project Folder

```bash
cd Weather-forecast
```

### Get Your OpenWeather API Key

1. Visit https://openweathermap.org/api
2. Create an account
3. Generate an API key

### Configure API Key

Open `script.js` and replace:

```javascript
const API_KEY = "your_api_key_here";
```

with your actual API key.

### Run the Application

Simply open:

```text
index.html
```

in your browser.

You can also use the VS Code Live Server extension for a better development experience.
